Node Modules:

  • Module is a set of functions. (think about js libraries such as Math).
  • 3 types of modules:
    1. Local Modules(own created modules)
    2. Build-in Modules(node.js own modules) - http, url, path, fs, (no need to install)
    3. External modules(Managed by npm)

Local modules export-import:

  1. exports.exportName = functionName; to use function from external file. এখানে সমান চিহ্ন এর আগের নাম দিয়ে ফাংশনটিকে অন্য জায়গা থেকে access করতে হবে। আর সমান চিহ্নের পরে function এর এক্সএক্ট নাম দিতে হবে।
  2. যে file থেকে function কে import করতে হবে, সে ফাইলে গিয়ে দুইটি কাজ করতে হবে।:
    1. const variableName = require(file directory); তাহলে ঐ ফাইলের সবকিছু একটি এই ভেরিএবল এর মধ্যে চলে আসবে।
    2. variableName.exportName(); তাহলে ফাংশনটিকে আমরা এই ফাইল থেকে ইউজ করতে পারবো।
    3. এই সেইম প্রসেসে সব ধরণের ডেটা টাইপের জিনিসই export করা যায়।
    4. তবে একাধিক function, variable কে export করতে হলে বার বার exports.exportName = functionName; না লিখে এভাবে লেখা যায়:
      module.exports = {function1, function2, ...., variable1, variable2....}
    5. তবে const variableName = require(file directory); এই প্রসেসে ঐ file এ যতগুলো function আছে, সবগুলোই চলে আসবে। কিন্তু আমরা যদি চাই শুধুমাত্র একটি বা দুইটি function আমাদের লাগবে সেক্ষেত্রে আমরা এভাবে করতে পারি const{function1, function2} = require('file directory'); তাহলে আমরা যতগুলো function চাইবো, ততগুলোই আসবে। এর বেশি আসবে না।
      আবার আমরা চাইলে function কে exports.functionName এভাবে ডিক্লারেশন করলে ও export হয়ে যাবে।

Built in modules- FS module:

FS module এ অনেকগুলো function বা method রয়েছে। যেমন

  1. writeFile() -> file তৈরি করতে বা file লেখতে।
  2. appnedFile() -> file আপডেট করতে।
  3. readFile() -> file কে রিড করতে।
  4. rename() ->file কে রিনেম করতে।
  5. Unlink() -> file কে ডিলিট করতে।
  6. Exists() -> কোনো একটি file আছে কী না তা চেক করতে।

তবে এই function গুলো হলো ‍asynthronous function. একই কাজ করার জ

Example of array object condition in a function:

const numbers = [1,2,3,4,5,6];
const sortNumbers = (array)=>{
let evenNum= [];
let oddNum =[];
const arrangedNum = {
even:[],
odd:[]
}
for (n of array){
if (n%2 ===0){
// evenNum.unshift(n)
evenNum.push(n)
}
else{
oddNum.push(n)
}
}
arrangedNum.even = evenNum;
arrangedNum.odd=oddNum;
return arrangedNum
}
console.log(sortNumbers(numbers));

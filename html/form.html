HTML Form

The HTML <form> is used to create a form

Form and its attributes

A form is essentially a container for inputs

The following code shows the basic structure of a form

<form>
The input elements goes header
</form>

  • action: Specifies the URL or file where the form data is sent. it also points to the back-end of our web page, which handles the form submition
  • method: Specifies the HTTP method for submitting the form(GET or POST). it also used to upload the data.

The Html form tag is required when you want to collect information that users provide.

For example. You may want to collect specific data from users, such as name, email address and passowrd.

The input tags

The <input> tag is used to create various types of input fields within a form.

Syntax:<input type="..." attribute="....">

Types of input fields:

  • Text:<input> type="text"
  • Password:<input> type="passowrd"
  • Email:<input> type="email"
  • Number:<input> type="number"
  • Checkbox:<input> type="checkbox"
  • Radio:<input> type="radio"
  • Submit Button<input> type="submit"

Input type Password

The <input> type="passowrd" defines a password field.

Input type submit

The <input> type="summit" defines a button for submitting form data to a server.

Input type radio

The <input> type="radio" defines a radio button. The radion button is used when only a single option needs to be selected out ot several offered options.

Input type checkbox

The <input> type="checkbox" defines a check button. The check button is used when multiple options needs to be selected out ot several offered options.

The <label> tag

The <label> tag represents a caption for an item in auser interface

The <textarea> tag

The <textarea> tag is used when the user has to enter details that may exceed a single sentence.

Syntax: <textarea> ... <textarea</>

Attributes:
  • Rows:Specifies the visible number of rows in the textarea
  • Cols:Specifies the visible number of columns in the textarea.

rows and cols attributes to allow you to specify an exact size for the <textarea> to take. setting these is a good idea for consistency, as browser defaults can differ.

HTML <audio> and <video> tag

The <audio> tag is used to embed audio fields.

The <video> tag is used to embed video fields.

Syntax: <audio src="audio.mp3 controls> ... <audio</>

Syntax: <vidio src="video.mp4 controls> ... <audio</>

Attributes:
  • src:Specifies the url or the path of the audio file.
  • controls:adds audio control(play, pause, volume) to the audio player .

rows and cols attributes to allow you to specify an exact size for the <textarea> to take. setting these is a good idea for consistency, as browser defaults can differ.

Select and option tag

  • The <select> element is most often used in a form, to collect user input
  • The name attribute is needed to reference the form data after the form is submited.
  • The id attribute is needed to associate the drop-down list with a label.
  • The <option> tags inside the <select> element define the available options in the dropdown list

Some Input tag attributes:

  • Value:<input value="John Doe"> This sets the default value of the input field to "John Doe"
  • required:<input required> This make the input field mandatory and the form cannot be submitted without entering a value.
  • disabled:<input disabled> This attibute disables the input field, making it non-editable and non-interactive.
  • readonly:<input readonly> makes the input field read-only, preventing users from editing its value
  • placeholder:<input placeholder="Enter your email"> provide a hint or example text "Enter your email" within the input field.

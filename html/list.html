Html List(ordered, unordered)

  • The <li> tag defines a list item
  • The <li> is used inside ordered lists(<ol>) and unordered lists(<ol>)
  • In <ul>, the list items will usually be displayed with bullet points.
  • In <ol>, the list items will usually be displayed with numbers or letters.

Ordered List Type

By default ordered list are numbered l, 2, 3 etc list. We can change order number with type property

type= "1"
  1. Bangla
  2. English
  3. ICT
type= "a"
  1. Bangla
  2. English
  3. ICT
type= "A"
  1. Bangla
  2. English
  3. ICT
type= "i"
  1. Bangla
  2. English
  3. ICT
type= "I"
  1. Bangla
  2. English
  3. ICT

Unordered List Type

By default ordered list are disc(bold dot) etc list. We can change unorder list style with type property

type= "disc"
  • Bangla
  • English
  • ICT
type= "circle"
  • Bangla
  • English
  • ICT
type= "square"
  • Bangla
  • English
  • ICT
type= "none"
  • Bangla
  • English
  • ICT

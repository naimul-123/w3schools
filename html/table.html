Html Table

Table has comumn and row. First row is table head and rest is table body. in body, there have many row contains data.

table tags

  • Table: table tag is used to create an html table to organize data in rows and column.
  • Table Head: thead
  • Table Body: tbody
  • Table Footer: tfoot
  • Table Row: tr represent a table row within the table
  • Table data or table cell: td represents a standard table data cell
  • Table header cell: th represent a standard table header cell
  • Table Caption: caption tag adds a caption or title to the table

table attributes

  • colspan Attribute: Specifies the number of columns a table cell should span
  • rowspan Attribute: Specifies the number of rows a table cell should span

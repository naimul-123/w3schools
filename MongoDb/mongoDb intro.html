what is mongodb?:

  • Document oriented database system.
  • Store data as JSON-like format.
    {
    name: "Naimul Islam",
    age: 31,
    language:["Bangla", "English"],

RDBMS Vs MongoDB terms

  1. Database = database
  2. Table = collection
  3. Rows = Documents
  4. Columns = Fields

Mongoshell commands

  1. test>
    help
  2. Shell Help:
  3. use->Set current database
  4. show->'show databases'/'show dbs': Print a list of all available databases.
  5. 'show collections'/'show tables': Print a list of all collections for current database.
  6. 'show profile': Prints system.profile information.
  7. 'show users': Print a list of all users for current database.
  8. 'show roles': Print a list of all roles for current database.
  9. 'show log ': log for current connection, if type is not set uses 'global'
  10. 'show logs': Print all logs.
  11. exit->Quit the MongoDB shell with exit/exit()/.exit
  12. quit->Quit the MongoDB shell with quit/quit()
  13. Mongo->Create a new connection and return the Mongo object. Usage: new Mongo(URI, options [optional])
  14. connect->Create a new connection and return the Database object. Usage: connect(URI, username [optional], password [optional])
  15. it->result of the last line evaluated; use to further iterate
  16. version->Shell version
  17. load->Loads and runs a JavaScript file into the current shell environment
  18. enableTelemetry->Enables collection of anonymous usage data to improve the mongosh CLI
  19. disableTelemetry->Disables collection of anonymous usage data to improve the mongosh CLI
  20. passwordPrompt->Prompts the user for a password
  21. sleep->Sleep for the specified number of milliseconds
  22. print->Prints the contents of an object to the output
  23. printjson->Alias for print()
  24. convertShardKeyToHashed->Returns the hashed value for the input using the same hashing function as a hashed index.
  25. cls->Clears the screen like console.clear()
  26. isInteractive->Returns whether the shell will enter or has entered interactive mode

Example of array object condition in a function:

const numbers = [1,2,3,4,5,6];
const sortNumbers = (array)=>{
let evenNum= [];
let oddNum =[];
const arrangedNum = {
even:[],
odd:[]
}
for (n of array){
if (n%2 ===0){
// evenNum.unshift(n)
evenNum.push(n)
}
else{
oddNum.push(n)
}
}
arrangedNum.even = evenNum;
arrangedNum.odd=oddNum;
return arrangedNum
}
console.log(sortNumbers(numbers));

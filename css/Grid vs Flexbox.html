Grid VS Flexbox

SL Grid Flexbox
01. Multi Dimensional layout One dimensional layout either column wise or row wise
02. Focuses on content placement layout First Focuses on content flow Content First
03. Provides a mechanism to divide available viewport for layout into columns and rows Widths (or heights) of flex items are determined by the content of the items.
04. Each item can be positioned anywhere on a grid cell. Spacing and aligning items are flexible.
05. Complex design is easier with grid More enargy needed to build complex design.

When to use Flexbox

  • You have a small design to implement.
  • You need to align elements.
  • You need a conten-first design.

What is Media Query?

  • Used to create different layouts depending on the size of the viewport.
  • Used to check width and height of the viewport.
  • Used to check orientation (is the tablet/phone in landscape or portrait mode?)

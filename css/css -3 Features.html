Core Features of css3

CSS3 introduced several new features and enhancements compared to its predecessor.

  • Selector:
    1. Attribute Selector: Allows you to select elements based on their attributes.
    2. nth-child and nth-of-type: Enables the selection of elements based on their position within a parent.
  • Box Model:
    1. Box-sizing Property: Allows you to specify whether an element's padding and border should be included in its total width and height.
    2. Box Shadows and Rounded Corners: Enables the creation of shadows around elements and the rounding of element corners
  • Text Effects:
    1. Text Shadows: Adds shadows to text.
    2. Text Overflow: Defines how overflowed content should be handled in a text container.
    3. Web Fonts: Allows the use of custom fonts in web pages.
    4. Transformations and Transitions:
    5. 2D and 3D Transformations: Translate, rotate, scale, and skew elements in two or three dimensions.
    6. Transitions: Smooth animations between different states of an element.
  • Animation:
    1. Keyframe Animations: Define custom animatins by specifying keyframes at various points in time.
    2. Animation Properties: Control the duration, timing, and other aspects of animations.
  • Flexbox and grid layout:
    1. Flexbox: Provides a flexible way to desing layouts, especially suitable for one-dimensional layouts.
    2. Grid Layout: Allows for more comple two-dimensional layouts
  • Media Queries:
    1. Responsive Design: Allows styles to be applied based on charcteristics of the device such as screen size, resolution or orientation.
  • Multi-column Layout:
    1. Column-count and Column-gap: Facilitates the creation of multi-column text layout.
  • Gradients and Backgrounds:
    1. Linear and Radial Gradients: Allows the creation of smooth color transitions.
    2. Mulitple Background Images: Apply multiple background images to an element.
  • Border Images:
    1. Border-image Property: Lets you use an image as a border for an element.
  • Opacity and RGBA:
    1. Opacity Property: Sets the transparency of an element.
    2. RGBA Colors: Allows the specificatin of colors using red, green, blue and alpha values.

CSS Layout

ওয়েবসাইটের বিভিন্ন ধরণের লেআউট থাকে। সিঙ্গেল কলাম লে আউট, স্টিকি হেডার লেআউট, কন্টেন্ট মেনুবার লেআউট।

লেআউট সম্পর্কে জানতে হলে সবার আগে CSS এর কিছু ইউনিট আমাদেরকে জানতে হবে। এগুলো হলো:

  • vh-equal to 1% of the height of the browser window size.
  • vw- equal to 1% of the width of the browser window size.
  • rem-relative to font-size of the root element.
  • em- Relative to the font-size of the parent element.
  • px-Pretty self explainatory. Absolute length in pixel.
  • %-Relative to the value of parent element. 100% is the width of the parent element.

Absolute vs Relative

  • Pixels(px)
  • Inches(in)
  • Centimeters(cm)
  • Millimeters(mm)
  • Points(pt)
  • Picas(pc)

Relative

  • Percentages(%)
  • Font size(em, rem)
  • Character sizes(ex, ch)
  • Viewport dimensions(vh, vw)
  • Viewport max(vmax)- use the largest of the viewport width or height.
  • Viewport min(vmin)-use the smallest of the viewport width or height

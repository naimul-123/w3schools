Javascript Function

What is function?:

Function is a block of code that received one or more input value that called parametter and returns a value based on parametter/parametters

ঘরের ছোট ছেলেকে যেমন সবাই যখন তখন যেকোনো কাজ করতে বলে তেমনি ফাংশন দিয়ে আমরা যখন তখন যেকোনো কাজ করিয়ে নিতে পারি।

Function এর একটি সুপার পাওয়ার আছে আর যেটা হল ফাংশন এর প্যারামিটারে ভ্যালু বসিয়ে বিভিন্ন ভ্যালু দিয়ে ফাংশনে বিভিন্ন রেজাল্ট পাওয়া যায়।

Function parameter:

এই মানগুলোর উপর ভিত্তি করে function আউটপুট দেয় সেগুলোকে Function parametter বলে। Function parametter, function এর first bracket এর মধ্যে কমা দিয়ে দিয়ে দিতে হয়। Function parametter এক বা একাধিক হতে পারে। তবে যতগুলো প্যারামিটার দেয়া হবে। Function parametter কোনো একটি ভ্যালু বা ভ্যারিয়েবলের মান যে কোন একটি হতে পারে। আর function call করার সময় parameter এর জায়গায় যা দিতে হয় সেটাকে বলে Argument.

Return of function:

exampel:
const add = (num1, num2)=>{
const result = num1+num2;
return result;
}
const sum = add(25, 20)
console.log(sum);

প্রসঙ্গত উল্লেখ্য, function এর শুধুমাত্র একটিই রিটার্ন হতে পারে। সেটা হতে পারে যেকোনো ভ্যালু তথা number, string, boolean, array, object etc.

Recap and Conditional Return:

আমরা চাইলে function এ কোনো কন্ডিশন দিয়ে ও কোনো ভ্যালু return করতে পারি। যেমন:
// return a value with condition
const isEven = (num)=>{
if (num %2 === 0){
return true;
}
else{
return false;
}
}
console.log(isEven(24));//returns true

Different types of parameters:

Function এর parameters যেকোনো ডেটা টাইপ তথা number, string, boolean, array and object হতে পারে।

Example of array as parameter of a function:
const numbers = [1,2,3,4,5,6];
const sum = (array)=>{
let result = 0;
for (n of array){
result = result+n
}
return result
}
const total = sum(numbers);
console.log(total)

Example of array object condition in a function:

const numbers = [1,2,3,4,5,6];
const sortNumbers = (array)=>{
let evenNum= [];
let oddNum =[];
const arrangedNum = {
even:[],
odd:[]
}
for (n of array){
if (n%2 ===0){
// evenNum.unshift(n)
evenNum.push(n)
}
else{
oddNum.push(n)
}
}
arrangedNum.even = evenNum;
arrangedNum.odd=oddNum;
return arrangedNum
}
console.log(sortNumbers(numbers));

function এর ভিতরে arguments নামে একটি জিনিস আছে যেটা function এর মধ্যে যতগুলো arguments আছে তা ‍array এর মতো করে object আকারে রিটার্ন করবে। এটি শুধুমাত্র function এর ভিতরেই কাজ করবে।

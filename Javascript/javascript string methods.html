Javascript String Method

String:

String এক ধরণের ডেটা। যা কোটেশন মার্ক বা ডাবল কোটেশন মার্ক দিয়ে লেখা হয়।

নিম্নোক্ত উপায়ে String Dicleration করা যায়।

  1. with single or double quotaition mark -- 'Rahim' or "Rahim"
  2. With back tick --`My string`
  3. With new keyword --new String("My string") এ ক্ষেত্রে string টি একটি object হয়ে যায়।
  4. Array এর মতো string এ length method ব্যবহার করা যায়। string.length-> string এর length রিটার্ন করবে। এক্ষেত্রে white space ও কাউন্ট হবে।
  5. Array এর মতো string এ ও index দিয়ে ‍string এর পজিশনভ্যালু বের করা যায়। ‍string[0] -> string এর ফার্স্ট পজিশনের ভ্যালু রিটার্ন করবে। তবে পার্থক্য হল array এর index দিয়ে যেভাবে ভ্যালুকে পরিবর্তন করা যায়, এখানে সেভাবে পরিবর্তন করা যায় না। এজন্য বলা হয় string is immutable.

String Comparision lowercase, uppercase and trim:

  1. string.toLowerCase() -> return lowercase value
  2. string.toUpperCase() -> return uppercase value
  3. string.trim() -> to remove white space from string
  4. string.trimStart() -> to remove white space from start position of string
  5. string.trimEnd() -> to remove white space from End position of string

String slice, Join, Concat and Includes

  1. string.slice(indexStart, indexEnd) -> slice দিয়ে array এর মতো string এ ও কাজ করা যায়। আমাদের ‍যদি কোনো string এর কোনো আংশিক পার্ট দরকার হয় তখন আমরা slice মেথডটি ব্যবহার করতে পারি। এক্ষেত্রে প্রথম ইনডেক্স থেকে শুরু হয়ে শেষ ইন্ডেক্স এর আগ পর্যন্ত অংশ আমাদেরকে রিটার্ন করবে। ‍slice এর কাছাকাছি হল substring()
  2. string.split(' separator ') -> return an array with separated values of string based on separator.
  3. split এর বিপরীত হল join . string কে ‍ split করে একটি array পাওয়া যায়, আবার array কে string এ convert করতে array.join('separator') এভাবে করে একটি string পাওয়া যাবে।
  4. string1.concat(string2, string3).concat(string4) ->to concat string1, string2, string3 and string4
  5. string.includes("value") returns true if string contains value, returns false if not contains

Reverse string:

String কে বিভিন্ন ভাবে reverse করা যায়।

  1. for of loop দিয়ে এভাবে করতে পারি।
    const myName="Naimul Islam";
    let reverseName = '';
    for(const letter of myName){
    reverseName= letter + reverseName;
    }
    console.log(reverseName);
  2. for loop দিয়েও করতে পারি।
    const myName="Naimul Islam";
    let reverseName = '';
    for(let i = 0; i< myName.length; i++ ){
    const letter= myName[i];
    reverseName= letter + reverseName;
    }
    console.log(reverseName)
  3. আবার চাইলে string.split('').reverse().join('') এভাবে ও করতে পারি।
    console.log(myName.split('').reverse().join(''));

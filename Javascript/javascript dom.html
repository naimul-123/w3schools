DOM intro:

DOM= Document Object Model. DOM এর মধ্যে ৩টি জিনিস থাকে।

  1. HTML for markup
  2. CSS for style
  3. Javascript for manipulate
DOM এ Html একটি tree এর মতো করে থাকে। যার মূল হল dom এবং html elements গুলো হল শাখা প্রশাখা।

What DOM Defines?

  1. The events for all HTML elements.
  2. The methods to access all HTML elements.
  3. The properties of all HTML elements as objects.
  4. All HTML elements as objects.

What can do Javascript with DOM?:

  1. Javascript can change all the html elements in the page.
  2. Javascript can change all the HTML attributes in the page.
  3. Javascript can change all the CSS styles in the page.
  4. Javascript can remove existing HTML elements and attributes.
  5. Javascript can add new HTML elements and attributes.
  6. Javascript can react to all existing HTML events in the page.

Get Elements By TagName:

আমরা HTML এর কোনো ইলিমেন্টসকে document.getElementsByTagName("tagName"); দিয়ে পেতে পারি। এক্ষেত্রে এটি একটি HTMLCollection return করবে। যেটি দেখতে অনেকটা array এর মতো। তবে পুরোপুরি array না। ‍array like object.

Get Element By Id:

HTML এর কোনো একটি ইলিমেন্টকে document.getElementById("idName"); দিয়ে পেতে পারি। এক্ষেত্রে এটি একটি নির্দিষ্ট html element কে return করবে। id ইউনিক হওয়ায় একটি html document-এ শুধুমাত্র একটি id ই ব্যবহার করা যায়।

Get Elements By ClassName:

আমরা HTML এর কোনো নির্দিষ্ট className এর ইলিমেন্টসগুলোকে document.getElementsByClassName("className"); দিয়ে পেতে পারি। এক্ষেত্রে এটিও একটি HTMLCollection return করবে। যেটি দেখতে অনেকটা array এর মতো। তবে পুরোপুরি array না। ‍array like object.

querySelectorAll:

HTML এর ইলিমেন্টসগুলোকে CSS মতো করে সিলেক্ট করতে এটি ব্যবহার করা হয়। document.querySelectorAll("tagName or #idName or .className"); দিয়ে পেতে পারি। এক্ষেত্রে এটি একটি NodeList return করবে। যেটিও দেখতে অনেকটা array এর মতো। তবে পুরোপুরি array না। ‍array like object. তবে আমরা যদি NodeList এর শুধু প্রথম ইলিমেন্টটিকেই পেতে চাই তাহলে document.querySelector("tagName or #idName or .className"); দিতে হবে।

getAttribute:

HTML এর ইলিমেন্টসগুলোর কোনো Attribute পেতে চাইলে যেকোনো ভাবে ইলিমেন্টটিকে সিলেক্ট করে তারপর document.querySelectorAll("tagName or #idName or .className").getAttribute("attributeName"); দিয়ে পেতে পারি। এক্ষেত্রে এটি ঐ ‍element এর ‍attribute গুলো রিটার্ন করবে।

class list:

HTML এর ইলিমেন্টসগুলোর সবগুলো class পেতে চাইলে যেকোনো ভাবে ইলিমেন্টটিকে সিলেক্ট করে তারপর document.querySelectorAll("tagName or #idName or .className"); দিয়ে পেতে পারি। এক্ষেত্রে এটি একটি array return করবে, যেখানে সবগুলো class থাকবে। classList এর কয়েকটি property আছে। যেমন:

  • classList.add("className"); to add class in selected elements.
  • classList.remove("className"); to remove class from selected elements.
  • classList.contains("className"); to check if the class name contains in selected elements.

setAttribute:

HTML এর ইলিমেন্টসগুলোতে কোনো একটি ‍attribute যোগ করতে চাইলে document.querySelectorAll("tagName or #idName or .className").setAttribute("attributeName", "attributeValue");

Add/change style with javascript:

আমরা জাভাস্ক্রিপ্ট এর মাধ্যমে চাইলে কোনো একটি ইলিমেন্ট এর স্টাইল দিতে পারি অথবা চেঞ্জ করতে পারি. এটা করতে হলে আমাদেরকে যেকোনো ভাবে ইলিমেন্টকে সিলেক্ট করতে হবে। সিলেক্ট করে তারপর .style.styleName = "style value"; এভাবে দিতে হবে। এক্ষেত্রে styleName এ যেখানে হাইফেন আছে, সেখানে হাইফেন উঠে গিয়ে বড় অক্ষরে হবে। যেমন সিএসএস এ background-color আছে এর পরিবর্তে হবে backgroundColor ।

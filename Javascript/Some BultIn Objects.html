Some built in objects in javascript

Math object:

Javascript has some built in object that contains some functions. one of them is Math. some math methods are...

  1. Math.min(numbers), returns smallest number of given numbers.
  2. Math.max(numbers), returns largest number of given numbers.
  3. Math.PI() returns PI value ;
  4. Math.abs(number), returns absolute value of a number.
  5. Math.round(number), returns integer value of a number. based on decimal value. if decimal value greater than or equal 5 then return upper integer else return lower integer value.
  6. Math.floor(number), returns lower integer value of a number.
  7. Math.ceil(number), returns upper integer value of a number.
  8. Math.random(), returns a number between 0 to 1.

Date object:

Some date object examples are..

  1. new Date(); returns today's date
  2. const date = new Date("yyyy-mm-dd"); to asign a date as a variable value
  3. We can also asign a varable of a date thus-> const date = new Date(year, monthIndex, dayNumber, hours, minute, second);
  4. date.getMonth() returns month index of given date. In javascript, month index start from 0. So Jonuary will be 0, February will be 1 etc.
  5. We can set a date variable's month or year or day like-> date.setMonth(11); to set date variable's month's value december.
  6. to convert date as local string format date.toLocalString(), if we want to get special region we can give a parameter like date.toLocalString("en-GB"); return date as English Great Britain format
  7. unix epoc -- 01 january 1970

Javascript swap

Swap বলতে বুঝায় অদল বদল। আমরা যদি দুইটি Variable এর ভ্যালুকে অদল বদল করতে চাই তাহলে সেটি কয়েকভাবে করা যায়।

  1. temporary varable এর মাধ্যমে।
  2. Destructurning method ex: [x,y] = [y,x] এর মাধ্যমে।
  3. অংক বা সংখ্যা থাকলে যোগ বিয়োগের মাধ্যমে।
  4. বিট ওয়াইজ ম্যানুপুলেশন এর মাধ্যমে।

Recap and Conditional Return:

আমরা চাইলে function এ কোনো কন্ডিশন দিয়ে ও কোনো ভ্যালু return করতে পারি। যেমন:
// return a value with condition
const isEven = (num)=>{
if (num %2 === 0){
return true;
}
else{
return false;
}
}
console.log(isEven(24));//returns true

Different types of parameters:

Function এর parameters যেকোনো ডেটা টাইপ তথা number, string, boolean, array and object হতে পারে।

Example of array as parameter of a function:
const numbers = [1,2,3,4,5,6];
const sum = (array)=>{
let result = 0;
for (n of array){
result = result+n
}
return result
}
const total = sum(numbers);
console.log(total)

Example of array object condition in a function:

const numbers = [1,2,3,4,5,6];
const sortNumbers = (array)=>{
let evenNum= [];
let oddNum =[];
const arrangedNum = {
even:[],
odd:[]
}
for (n of array){
if (n%2 ===0){
// evenNum.unshift(n)
evenNum.push(n)
}
else{
oddNum.push(n)
}
}
arrangedNum.even = evenNum;
arrangedNum.odd=oddNum;
return arrangedNum
}
console.log(sortNumbers(numbers));

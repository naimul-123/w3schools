Javascript Array

What is array:

Array এক ধরণের কন্টেইনার যা একাধিক ভ্যালু স্টোর করতে পারে।

Array Syntax
const arrayName =[value1, value2, value3, .........]

Array Index 0 থেকে শুরু হয়. index দিয়ে এরের নির্দিষ্ট ইন্ডেক্সের ইলিমেন্টের ভ্যালু পাওয়া যাবে। আবার ইনডেক্স দিয়ে সেই ইনডেক্সের ভ্যালু সেট করা ও যায়।

Array Methods:

  1. array.length -> to find how meny elements have in the array
  2. array.includes(value) if an array contains the value than return true else return false
  3. array.indexOf(40)--to find index number of an array first same elements. it returns only first element's index if duplicate value contain. যদি ভ্যালু না থাকে তাহলে -1 রিটার্ন করবে।
  4. Array.isArray(arrayName) --return true if arrayName is an array else return false.
  5. array.join('seperator') to join elements of an array with seperator
  6. array.conact(array2) return a new array with concatinated two array.
  7. array.push(newValue1, newValue2, newValue3.... )- to add new values to the last of array
  8. array.pop() --to delete the last element of an array returns deleted value.
  9. array.shift() -- to delete the first element of an array returns deleted value.
  10. array.unshift(newValue1, newValue2, newValue3, .....) -- to add new values to the first of an array.
  11. array.length-1 -- to find last element index of an array
  12. array.slice(indexStart, indexEnd) return a new array প্রথম ইনডেক্স থেকে শুরু হবে কিন্তু লাস্ট ইন্ডেক্সের আগ পর্যন্ত নিবে।
  13. array.splice(startIndex, deleteCount, newElement1, newElement2 ...) to delete some value and add new value.

Array Loop:

আমরা এর আগে জেনে আসছিলাম, for of loop এরেতে ব্যবহার হয়। for of loop ছাড়াও আরো কয়েকভাবে array এর মধ্যে loop করা যায়। তবে বিভিন্ন কাজের সুবিধার্থে আমাদেরকে array এর index কে ও দরকার হবে। সেক্ষেত্রে for of loop দিয়ে আমাদের কাজ হবে না। for loop দিয়ে কাজ চালানো গেলেও অনেক কোড বেশি লিখতে হয়। এজন্য array এর কয়েকটি বিল্ডইন লুপ মেথড আছে।

Array Reverse

Array কে বিভিন্ন ভাবে reverse করা যায়। যথা:-

  1. array.reverse() method এর মাধ্যমে।
  2. loop এর মধ্যে array.unshift দিয়ে
  3. loop কে উল্টো ভাবে চালিয়ে array.push মেথডের মাধ্যমে

Sort an array:

Sort দুই ভাবে হয়।

  1. Ascending order. Ex: a,b,c,d...
  2. Descending order. Ex: z,y,x...
array.sort() মেথড দিয়ে এটা করা যায়। তবে javascript alphabetical order এ শর্ট করে তাই এভাবে number কে ‍sort করা যায় না। number কে ‍sort করতে হলে sort method এর মধ্যে একটি callback Function নিতে হবে। যেটা এই প্রবলেমকে ফিক্সড করবে। Ascending করতে হলে-- ‍
array.sort((a,b)=> {return a-b});
Descending করতে হলে-- ‍
array.sort((a,b)=> {return b-a});

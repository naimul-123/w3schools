জাভাস্ক্রিপ্ট প্রাথমিক আলোচনা

পৃথিবীর একটি পাওয়ারফুল প্রোগ্রামিং ল্যাংগুয়েজ হল জাভাস্ক্রিপ্ট। আগে ওয়েবসাইটগুলো ছিল অনেকটা বইয়ের মতো। কিছু রিচার্সের কাজে ওয়েবসাইট ব্যবহার হলেও এর কোনো ইউজার ইন্টারএক্টিভিটি ছিল না। অর্থাৎ ইউজার ওয়েবসাইট শুধু দেখতেই পারতো। ওয়েবসাইটে কোনো কিছু পরিবর্তন বা তার কোনো মতামত দিতে পারতো না। এরপর ডেভেলপাররা চিন্তা করলো কিভাবে ওয়েবসাইটগুলো সাধারন মানুষের আরো অনেক কাজ করার কাজে ব্যবহার করা যায়। এই চিন্তা থেকেই ব্রেন্ডন এইচ ১৯৯৫ সালে মাত্র ১০ দিনে জাভাস্ক্রিপ্ট তৈরি করেন। পরে ১৯৯৭ সালে এটিকে অফিসিয়াল স্ট্যান্ডার্ড একমাতে রেজিস্ট্রার্ড করে। আর এর অফিসিয়াল নাম হয় একমা স্ক্রিপ্ট।

জাভাস্ক্রিপ্ট ইউজ করার উপায়

আমরা এর আগে সিএসএস এর ব্যবহার শিখেছি। সিএসএস এর মতো জাভাস্ক্রিপ্ট ও ৩ ভাবে ব্যবহার করা যায়।

  1. ইনলাইন জাভাস্ক্রিপ্ট
  2. স্ট্রিপ্ট ট্যাগের মাধ্যমে
  3. এক্সটার্নাল সোর্স এর মাধ্যমে। এ ক্ষেত্রে স্ক্রিপ্ট ট্যাগের সোর্সে ফাইলের নাম লিখে দিতে হবে।

জাভাস্ক্রিপ্ট হাই লেভেল ল্যাংগুয়েজ। যেসকল প্রোগ্রামিং ল্যাংগুয়েজ মানুষ ও কম্পিউটার উভয়েই কিছু কিছু বুঝতে পারে সেগুলোকে হাই লেভেল ল্যাংগুয়েজ।

জাভাস্ক্রিপ্ট ইন্টারপ্রিটেড ল্যাংগুয়েজ

জাভাস্ক্রিপ্ট আমরা তিনটি জায়গায় ব্যবহার করবো।

  1. কনসোলে
  2. ভিএস কোড ইডিটরের কমান্ডপ্রম্ট এ
  3. কোড পেন এ

জাভাস্ক্রিপ্ট এর জন্য কতটুকু ম্যাথ জানা লাগবে?

  • সংখ্যা সম্পর্কে ধারণা রাখতে হবে। পজিটিভ নেগেটিভ, পূর্ণসংখ্যা, অপূর্ণ সংখ্যা, দশমিক সংখ্যা, জোড় সংখ্যা বেজোড় সংখ্যা ইত্যাদি।
  • ম্যাথের সাধারণ অপারেশনগুলো তথা যোগ, বিয়োগ, ভাগ, গুণ, ভাগশেষ, গড়, বড় সংখ্যা, ছোট সংখ্যা ধারা, ক্ষেত্রফল, প্রাইম নাম্বার, পারমুটেশন, কম্পিনেশন, গ্রাফ ইত্যাদি সম্পর্কে ধারণা রাখতে হবে।

জাভাস্ক্রিপ্ট ভেরিয়েবল

আমরা যখন কোনো জিনিস কোনো একটা পাত্রে রাখি তখন ঐ পাত্রে একটা নাম দিয়ে রাখি যাতে সহজেই সে জিনিসটা চিনতে পারা যায়। এর থেকেই ভেরিয়েবল এর ধারণাটা আসে। প্রোগ্রামিং এর ক্ষেত্রে আমাদের বিভিন্ন তথ্য বা ডেটা স্টোর করে রাখতে হয়। সেগুলোকে আমরা একটা ভেরিয়েবল এর মধ্যে রাখতে পারি। ভেরিএবল এর মানে হল কোনো একটা জিনিসের মান পরিবর্তন যোগ্য।

ভেরিএবল ডিক্লারেশন এর জন্য পাঁচটি জিনিস লাগে।

  1. Dicleration keyword-> for Javascript var, let, const
    var-> value changable anywhere
    let-> value changable only outside block. that means it's value accessable in only block area
    const -> value can be changable only with non-premitive data -> array, object. but can not changable with another array or object.
  2. Variable Name
  3. Equal sign
  4. value
  5. semicolon -> optional for Javascript.

জাভাস্ক্রিপ্ট ডেটা টাইপ

জাভাস্ক্রিপ্ট এ ৫ ধরণের ডেটা টাইপ আছে।

  1. Number-> Numirec value->1,2,3,...etc
  2. String ->text value ->"Hello world"
  3. Boolean-> true/false
  4. Null-> Empty variable
  5. Undefined-> Variable diclared, but still not assign value.

ভেরিএবল এর নামের আগে type of লিখে দিলে ভেরিএবল টাইপ পাওয়া যাবে।

জাভাস্ক্রিপ্ট ভেরিএবল ভ্যালু লেখার কিছু নিয়ম

  1. যদি নাম্বার টাইপ ভ্যালু হয় তাহলে শুধু নাম্বারটা লিখবো
  2. যদি স্ট্রিং টাইপ ভ্যালু হয় তাহলে সিঙ্গেল বা ডাবল কোটেশনের মধ্যে স্ট্রিং লিখবো।
  3. বুলিয়ান টাইপ হলে শুধুমাত্র true/false

জাভাস্ক্রিপ্ট ভেরিএবল নাম লেখার কিছু নিয়ম

  1. কোনো কিওয়ার্ডকে নাম হিসেবে ব্যবহার করা যাবে না।
  2. কোনো গ্যাপ বা স্পেস দেয়া যাবে না। আন্ডার স্কোর[স্নেক কেইজ] বা সবগুলো ওয়ার্ডের প্রথম অক্ষর বড় হাতের[প্যাসকাল কেজ] দেয়া যাবে, তবে জাভাস্ক্রিপ্টএ প্রথম ওয়ার্ডের প্রথম অক্ষর ছোট হাতের,আর বাকী সবগুলোর প্রথম অক্ষর বড় হাতের দেয়া[ক্যামেল কেজ] রিকমান্ডেড।
  3. কোনো কোটেশন দেয়া যাবে না।
  4. কিছু স্পেশাল ক্যারেকটার আছে যেগুলো ব্যবহার করা যাবে না।
  5. নাম্বার দিয়ে শুরু করা যাবে না। তবে প্রথম লেটার বাদে অন্য যেকোনো জায়গায় দেয়া যাবে।
  6. ভেরিয়েবল নাম কেস সেনসিটিভ।

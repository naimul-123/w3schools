Javascript Loop

Diffrent type of loop:

loop বলতে কোনো একটি কাজ একাধিকবার করা বোঝায়।

  1. for of loop
  2. while loop
  3. for loop
  4. do while loop

for of loop:

Array অথবা string এর প্রত্যেকটি উপাদান নিয়ে কাজ করার জন্য for of loop ব্যবহার হয়।

const numbers = [1,5,6,8,10];
for(num of numbers){
console.log(num);
}
returns numbers array element one by one

While loop:

কোনো একটি কাজ একটি নির্দিষ্ট শর্ত সত্য থাকা অবধি চলতে থাকবে এরূপ ক্ষেত্রে while loop ব্যবহার হবে।

let num = 0;
while(num <5){
console.log(`Loop until number become ${num}`)
num++
}

এখানে num এর ভ্যালু 5 না হওয়া অবধি এই লুপটি চলতে থাকবে। while loop এ সাধারণত ইনিশিয়াল ভ্যালু ডিফাইন করা থাকে না।

For loop:

for loop while loop এর মতোই। পার্থক্য হল syntax লেখাতে।

for(let num = 1; num<=10; num++){
console.log(num)
}

loop break:

loop এ আমাদের যদি একটা নির্দিষ্ট স্টেজে এসে থেমে যাওয়ার প্রয়োজন হয়, তাহলে আমরা break এর মাধ্যমে লুপটাকে থামিয়ে দিতে পারি।

for(let num = 1; num<=10; num++){
console.log(num)
if(num>5){
break;
}
}

এখানে num এর ভ্যালু যখন 5 এর বড় হয়ে যাবে, তখন লুপটি থেমে যাবে।

loop continue:

continue -> skip rest of the code for this iteration. একটি নির্দিষ্ট কন্ডিশন সত্য হলে লুপটি পরবর্তী কোডে না গিয়ে আবার শুরুতে চলে যাওয়া।

for(let num = 1; num<=10; num++){
if(num%2===1){
continue;
}
console.log(num)
}

এখানে যখন num%2===1 সত্য হবে তখন লুপটি নিচের কোডকে এক্সিকিউট না করে পুনরায় শুরুতে চলে যাবে।

do while loop:

normal while loop শর্ত সত্য না হলে চলবে না। কিন্তু do while loop আগে কোড এক্সিকিউট করে তারপরে শর্ত চেক করে। যদি শর্ত সত্য হয় তাহলে আবার চলে।

let num = 0;
do{ console.log(`Loop until number become ${num}`)
num++
} while(num <5){
}

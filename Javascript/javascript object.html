Javascript Object

Object:

Object হল non-primitive ডেটা টাইপ। প্রায় সব প্রোগ্রামিং ল্যাংগুয়েজেই অবজেক্ট আছে।

We can define object thus: An unordered collection of key-value pairs. Each key-value pair is called a property.

নিচে Object এর কিছু বৈশিষ্ট দেয়া হল:

  1. Object সাধারণত const দিয়ে ডিক্লেয়ার করা হয়।
  2. Object কার্লি ব্রেস {} এর মধ্যে লেখা হয়।
  3. Object এ একটি property ও একটি value দিয়ে ডেটা স্টোর করা হয়।
  4. Object এর মধ্যে সব ধরণের ডেটা টাইপ তথা string, number, boolean, null, undefined, array এমনকি object ও ভ্যালু হিসেবে স্টোর করা যায়।
  5. Difference between object and array :
    • Array stores data in ordered(sequential) collection but Object stores data in unordered collection.
    • Data can be accessed in array using numerical index. for object, values are accessed calling keys.

Object Declaretion:

Object কে নিম্নোক্ত ভাবে declare করা যায়।

  1. Using an object literal. const objectName = {key:value, key:value.....}
  2. Using new keyword const objectName = new Object(key:value, key:value.....)
  3. Using Obeect.create const objectName = Object.create(key:value, key:value.....)
  4. Using class or function-- Define an object constructor. then create objects of the constructed type.

Object Properties:

Object এর কোনো একটি property এর ভ্যালু এক্সেস করতে বা নতুন ভাবে সেট করতে dot notation ও bracket notation ব্যবহার করা হয়।

  1. with dot notation -> objectName.propertyName
  2. with bracket notation -> objectName[propertyName] এক্ষেত্রে property name কে একটি স্ট্রিং হিসেবে লিখতে হবে। অর্থাৎ সিঙ্গেল বা ডাবল কোটেশনের মধ্যে লিখতে হবে।
  3. কোনো একটি ভেরিএবল এর ভ্যালু হিসেবে property এর নামকে সেট করে তারপর সেই ভেরিএবলকে bracket notation এর মধ্যে বসিয়ে দিয়েও property এর ভ্যালু এক্সেস করা যায়।

তবে propertyName একাধিক ওয়ার্ডে হয় বা _, বা এরকম কিছু স্পেশাল কী দিয়ে শুরু হলে dot notation দিয়ে property এর ভ্যালু এক্সেস বা সেট করা যায় না। সেক্ষেত্রে bracket notation হল একমাত্র সমাধান।

Keys, Values, nested objects and delete

  1. কোনো একটি Object এ কতগুলো কী বা প্রপার্টি আছে তা দেখতে চাইলে Object.keys(ObjectName) returns an array with object property Name.
  2. কোনো একটি Object এর যতগুলো ভ্যালু আছে তা বের করতে চাইলে Object.values(ObjectName) returns an array with object values.
  3. nested object এর value বের করতে যতবার নেস্টেড ততবার dot notation বা bracket notation ও প্রপার্টি নেম দিতে হবে।
  4. object এর কোন প্রপার্টি ডিলিট করতে চাইলে delete objectName.propertyName দিতে হবে।
